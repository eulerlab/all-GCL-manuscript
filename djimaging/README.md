# Installation and Usage Instructions for All-GCL Dataset Processing

This code defines the DataJoint schema used to process the All-GCL dataset.
To use it you also need to download install the correct version of djimaging, which you can find here:
https://github.com/eulerlab/djimaging/releases/tag/all-gcl-v0.1.0

After installing djimaging, copy this code into the ``djimaging/djimaging/user/`` folder such that the path looks like this:
``djimaging/djimaging/user/all_gcl/``.
